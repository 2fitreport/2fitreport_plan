# 2FitReport Plan - 일정 관리 시스템

모던한 달력 인터페이스를 가진 일정 관리 시스템입니다. Notion 스타일의 미니멀 디자인과 강력한 기능을 제공합니다.

## 🎯 주요 기능

### 📅 달력 인터페이스
- 월별 달력 보기
- 각 날짜별 일정 표시 (색깔로 상태 표시)
- 월/년도 선택 모달
- 오늘 날짜 빠른 이동 버튼
- 상태별 색깔 범례 (진행중, 검수, 완료, 보류)
- 현황 요약 (각 상태별 개수)

### ✍️ 일정 관리
- **생성**: 제목, 날짜, 내용 입력으로 새 일정 등록
- **읽기**: 일정별 상세 정보 조회
- **수정**: 제목, 날짜, 내용 실시간 수정
- **삭제**: 일정 삭제 (확인 모달)

### 📊 상태 관리
- **진행중** (노란색): 새로 등록된 기본 상태
- **검수** (파란색): 진행중에서 변경 가능
- **완료** (초록색): 최종 완료 상태
- **보류** (빨간색): 일시 보류 상태

### 💬 댓글 기능
- 각 일정마다 댓글 작성 가능
- 댓글 개수 표시 (게시판 목록)
- 댓글 삭제 기능
- 댓글 작성 시간 표시

### 🎨 사용자 인터페이스
- Notion 스타일 미니멀 디자인
- CSS Modules 기반 스타일링
- 반응형 레이아웃 (모바일 지원)
- 부드러운 애니메이션
- 확인 모달 (상태 변경, 삭제, 등록)
- 수정 모달 (인라인 수정)

## 🛠 기술 스택

### Frontend
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules
- **State Management**: React Hooks

### Backend
- **Database**: Supabase PostgreSQL
- **Authentication**: Supabase Auth
- **Client**: Supabase JS Client

## 📁 프로젝트 구조

```
app/
├── components/          # 재사용 가능한 컴포넌트
├── date/               # 날짜별 게시판
├── posts/              # 게시물 관리
├── page.tsx            # 메인 달력 페이지
└── layout.tsx          # 레이아웃
lib/
└── supabase.ts         # Supabase 클라이언트
```

## 🚀 시작하기

### 설치
```bash
npm install
```

### 환경 설정
`.env.local` 파일 생성:
```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
npm run start
```
